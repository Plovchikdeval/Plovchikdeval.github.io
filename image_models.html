Модели для Генерации Фото

Модели для Фото

  • dalle-3: Генерация изображений на основе текстовых запросов.
  • sdxl: Модель для генерации изображений с высоким качеством.
  • sd-3: Генерация изображений с использованием Stable Diffusion.
  • flux: Модель для создания изображений с уникальными стилями.
  • flux-pro: Модель с улучшенной производительностью для создания изображений.
  • flux-anime: Модель для создания изображений в стиле аниме.
