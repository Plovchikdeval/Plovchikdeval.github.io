Текстовые Модели ИИ

Модели OpenAI

  • gpt-3: Базовая языковая модель третьего поколения OpenAI.
  • gpt-3.5-turbo: Быстрая и оптимизированная версия GPT-3.5.
  • gpt-4: Продвинутая версия GPT-4, обеспечивающая лучшую точность.
  • gpt-4o: Оптимизированная версия GPT-4 для меньшего потребления ресурсов.
  • gpt-4o-mini: Мини-версия GPT-4o, предназначенная для малых систем.
  • gpt-4-turbo: Ускоренная версия GPT-4 с меньшими затратами на вычисления.

Модели Meta

  • llama-2-7b: Модель LLaMA второго поколения с 7 миллиардов параметров.
  • llama-3-8b: Модель третьего поколения с 8 миллиардами параметров.
  • llama-3.1-8b: Модель для обработки сложных текстовых задач.
  • llama-3.1-70b: Мощная модель LLaMA третьего поколения с 70 миллиардами параметров.
  • llama-3.1-405b: Модель с 405 миллиардами параметров для самых сложных задач.

Модели Mistral

  • mixtral-7b: Модель с 7 миллиардами параметров для широкого спектра задач.
  • mixtral-8x7b: Мощная модель с улучшенной производительностью.
  • mistral-tiny: Легковесная модель для малых систем.
